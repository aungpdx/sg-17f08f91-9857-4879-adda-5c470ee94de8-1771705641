# The Olive Branch Australia (TOBA)

A modern Islamic charity platform providing Qurban, Aqiqah, Sadaqah, Zakat, and Waqf services for Muslims worldwide.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
npm install
npm run dev
```
